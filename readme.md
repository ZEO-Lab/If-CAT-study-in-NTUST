## 背景故事 Introduction ##
在一個平行時空的世界裡，台科校園裡的貓咪夢想著成為頂尖學者，她踏入了台灣科技大學，開始了艱辛卻充滿樂趣的大學生活。  
每個學期，小咪都會遇到不同的挑戰，玩家必須利用智慧和策略，通過各種課程要求，並在期末考中脫穎而出。  
最終，玩家將見證小咪順利畢業，邁向她的學術夢想。  
加入這場智慧與努力的冒險，幫助小咪戰勝每一個學期的考驗吧！
